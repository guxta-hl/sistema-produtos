# Sistema de Cadastro de Produtos

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- Java Swing
- MySQL
- JDBC

## Time de Desenvolvedores
- Sistemas LTDA

## Objetivo do Software
O sistema tem como objetivo permitir o cadastro e gerenciamento de produtos e seus respectivos preços, facilitando a organização e consulta dessas informações.

## Funcionalidades do Sistema
- Cadastro de produtos (nome e preço)
- Listagem de produtos cadastrados
- Ediçao de produto
- Exclusão de produtos
